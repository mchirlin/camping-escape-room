# Game Mastering — Notes

Project: 2026-camping-minecraft
Created: 2026-04-18

## Decisions

### Role
- Just dad running the game — no character costume
- NPC-style interactions: verify pickaxe at the mine, hand out items at crafting table, control soundboard

### Solo Operation
- No helper adult — running it all yourself
- Stay near the crafting table (in the tent) as home base
- Walk with kids to key moments (mine entrance, creeper, dig site) as needed

### Hints
- Just walk over and nudge them if stuck
- Keep it casual — no formal hint system needed for friends and family

### Intro Speech Ideas
- **Diary approach (recommended):** Use the diary you already own as an explorer's journal. Read the last entry aloud — it describes the Ender Dragon egg sighting near the campsite. Last page has Recipe Card #1 (pickaxe) tucked inside. Hand them the diary and a bag for collecting blocks. "Go!"
- **Simple version:** Gather kids, 30-second pitch: "There's an Ender Dragon egg buried somewhere at this campsite. You need to craft tools to find it. Here's your first recipe." Hand out Recipe Card #1.
- **Chest version:** Diary + Recipe Card #1 locked in a chest with a key hidden nearby. Mini-puzzle to kick things off.

### Running Twice
- Want to run it for two groups
- Reset plan:
  - Take blocks back after each craft (already planned)
  - Re-stack creeper boxes (pre-scored for easy reassembly)
  - Re-hide blocks in same spots
  - Re-bury dig container (prep two dig sites to avoid re-digging)
  - Reload recipe scrolls
