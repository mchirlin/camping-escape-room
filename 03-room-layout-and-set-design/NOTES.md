# Room Layout & Set Design — Notes

Project: 2026-camping-minecraft
Created: 2026-04-18

## Campsite Layout

```
[Bathroom/Mine] — up the hill with stones
        |
   [Campsite + Tent (crafting table inside)]
        |
   [Tiny stream — fishing spot, gold blocks nearby]
        |
   [Forest entrance]
        |
   [Trail → Larger stream]
```

## Decisions

### Crafting Table
- Inside the tent — home base, sheltered from weather
- Kids keep returning here between explorations

### Biome Locations
- **Forest biome** (wood, sticks): near trees around campsite and forest entrance
- **Mountain/Mine biome** (iron, string, diamonds): bathroom up the hill + stone pile area
- **River biome** (gold): in/near the tiny stream
- **Trail/deep forest**: creeper location, dig site

### Creeper Location
- Along the trail in the forest — kids won't go deep into the forest until the sword clue sends them there
- Hidden from initial view so they don't mess with it early

### Dig Site
- In the forest, off the trail
- Soft ground, pre-bury waterproof container before the game
- Prep two dig sites if running the game twice

### Decorations
- Minimal — one or two Minecraft-style biome signs (pixel font)
- Let the natural setting do most of the work

### Bathroom/Mine
- One dedicated bathroom stall with door to outside
- Only group using the facilities — no conflict with other campers
- Sign on door: "⛏ Abandoned Mine — Wooden Pickaxe Required"
- Blocks hidden inside on ledges, behind fixtures, in corners
